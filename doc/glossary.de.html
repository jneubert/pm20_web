PM20-Glossar

Systematikstelle
Eintrag in der Systematik (oder auch: Klassifikation), z.B. der Ländersystematik. Systematikstellen werden durch ihren Typ und eine Signatur definiert.
Systematik-Typ
Bezeichnet die jeweilige Systematik
geo - Ländersystmatik
subject - Sachsystematik
ware - Warensystematik
Signatur
(oder auch: Notation) Kurzer alphanumerischer Code für eine Stelle innerhalb einer Systematik, z.B. “A10” für Deutschland.
